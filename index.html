My Projects

EDUMOTION

An Angular built, Firebase hosted, front-end application that focuses on helping teachers in the classroom log students emotional state after each classroom activity in order to track emotional success. With this project I utilized CRUD functionality within a single-page application to maintain ease-of-use for teachers but still give the users ability to perform complex actions like editing, posting, retreving and analyzing data. Overall, EduMotion aims to provide teachers with the ability to reduce the hypothesis space and form higher granulairty into student observations.

    TECHNOLOGIES USED:
  • Firebase Database & API
  • JQuery
  • Angular 1.6
  • Adorable Avatars API
  • JavaScript
  • Chart JS

About

Why software design and development?
To answer this question is easy....I was made to do this!
At a young age I embarked on a journey to become a New York Times best-selling author and a world-renowned music journalist but after countless nights of editing articles and breaking wires on my drum kit, I found myself doing more debugging, command line trial and error and dealing with floppy-disk drama then what I originally had set out to do. I came to the realization that my true talents can be found in the inner workings of a computer screen. Writing has remained a real passion of mine but now has been transformed into a different language, a language primarily written in Python, Javascript and Angular. I've gone from telling stories about complex software programs built by some of the best developers on the market to designing code of my own built off of the lessons I have learned from my previous coworkers as well as from the amazing teachers I've turned to while attending NSS. I can now say I am a true craftsman of eloquent code written in a language that paints pictures in data.
